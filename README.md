# Value-Compass-Automization

## Step 1
- git clone https://github.com/Qyuan1223/Value-Compass-Automization.git

## Step: 2
- 从github拉去最新版本的项目：```git pull origin main```
- ！！！ 每次开始工作之前都最好先pull一下最新版本

## Step：3
- 查看当前项目状态：```git status```
- 将改动加入到github：```git add .```
- 查看加入后项目状态：```git status```
- 将add后的文件commit到github：```git commit -m "update dataset(name)"```
- 查看commit后项目状态：```git status```
- 将所有改动push到github上：```git push origin main```
